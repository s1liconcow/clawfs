OSS CLI

Mount A Local ClawFS Workspace

Install + Inspect

curl -fsSL https://clawfs.dev/install.sh | bash clawfs --help

Use the public CLI to inspect flags, configure a backing store, and mount a workspace locally.

First Mount

clawfs --mount-path /tmp/clawfs-mnt \ --store-path ~/.clawfs/store \ --local-cache-path ~/.clawfs/cache

That gives you the open-source filesystem path directly, without the hosted account workflow.

Use the hosted app for managed access, or run the OSS CLI directly.

Both paths start from the same install page, but the public repo is now focused on the filesystem itself.
